
# منصة "صياغة" — منصة ذكية لصناعة المحتوى التسويقي العربي

## نظرة عامة
منصة ويب عربية بالكامل (RTL) تساعد التجار والمسوقين في السوق السعودي والخليجي على توليد محتوى تسويقي احترافي باستخدام الذكاء الاصطناعي، مع نظام نقاط مرن وباقات اشتراك.

## التصميم والهوية
- **الاتجاه:** RTL عربي بالكامل، خط Tajawal/IBM Plex Arabic
- **الأسلوب:** بسيط ونظيف، مع دعم الوضع الفاتح والداكن
- **الألوان:** لوحة هادئة احترافية (أزرق نيلي + أبيض/داكن + لمسة ذهبية للعناصر المميزة)
- **التجاوب:** يعمل على الجوال والتابلت والديسكتوب

- شهادات وأمثلة
- CTA للتسجيل

- **المدخلات:** رابط منتج / وصف خام / نص سابق / فكرة / اختيار منتج محفوظ

- سجل المفضلة والقوالب المحفوظة في صفحة خاصة
- إمكانية إعادة التوليد أو التعديل

### 7. ملفات المنتجات (Product Profiles)
- إنشاء بطاقة منتج: اسم، وصف، سعر، مزايا، جمهور، كلمات محمية، نبرة، قوالب مفضلة

### 8. إعدادات الحساب
- Brand Voice Profile: تحديد نبرة العلامة الافتراضية
- Brand Terms Lock: قائمة المصطلحات الممنوعة من التغيير
- اللهجة الافتراضية والمنصة المفضلة

### 9. الباقات والنقاط
- عرض الباقات الثلاث (49/89/199 ريال) مع تفاصيل النقاط (350/900/2400)

- تنبيه عند انخفاض الرصيد مع زر "ترقية الباقة"

### 10. لوحة الأدمن
- إدارة طلبات التفعيل (عرض إثبات التحويل → قبول/رفض → شحن النقاط)
- إدارة المستخدمين ورصيدهم
- إدارة القوالب العامة (المُضافة مسبقاً)
- إحصائيات عامة

## البنية التقنية

### الذكاء الاصطناعي
- **Lovable AI Gateway** عبر server functions
- **Gemini Flash** للعمليات الخفيفة (3-5 نقاط): اختصار، تدقيق، إعادة صياغة
- **Gemini Pro** للعمليات الاحترافية (12-30 نقطة): تحليل، توليد متقدم، خطط

### تحليل الروابط
- Server function تجلب محتوى الصفحة (fetch + parse HTML)
- إذا فشل الجلب: يظهر حقل للصق اليدوي

### قاعدة البيانات (Lovable Cloud / Supabase)
- **profiles:** بيانات المستخدم الأساسية
- **user_roles:** أدوار (admin/user)
- **points_balance:** رصيد النقاط
- **transactions:** سجل العمليات والنقاط المستهلكة
- **subscription_requests:** طلبات الدفع اليدوي (حالة، إثبات، باقة)
- **product_profiles:** بطاقات المنتجات المحفوظة
- **saved_templates:** القوالب والمفضلة
- **brand_settings:** نبرة العلامة + المصطلحات المحمية
- **generated_content:** المخرجات المحفوظة
- **system_templates:** القوالب العامة (يديرها الأدمن)
- RLS على كل الجداول

### الأمان
- حماية كل الـ server functions بـ middleware مصادقة
- خصم النقاط على السيرفر فقط (لا يمكن التلاعب من العميل)
- التحقق من الرصيد قبل كل عملية
